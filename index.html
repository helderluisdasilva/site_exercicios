Linguagens Front End

O HTML5

O HTML5 (HyperText Markup Language) é uma linguagem de marcação semântica utilizada para definir o conteúdo de um site através de marcadores chamados de tags, essas tags possuem a seguinte estrutura <NomeDaTag>, com elas podemos definir o tipo de conteúdo de cada parte do nosso site, por exemplo, podemos definir um parágrafo usando a tag <p>, adicionar uma imagem usando a tag <img>, além de outros tipos diversos de conteúdo.

O CSS3

O CSS3 (Cascading Style Sheets) é uma linguagem utilizada em conjunto com o HTML para definir a estrutura do site e sua aparência, com ela definimos as propriedades e aparência de cada elemento do nosso site, suas medidas, cores, posicionamento, entre outras configurações, em outras palavras, o CSS é reponsável por dar vida a uma página web.

O JavaScript

O JavaScript é uma linguagem de programação utilizada para adicionar interatividade em um site, com ele podemos fazer a validação de formulários, criar menus complexos, além de muitos outros efeitos que não conseguimos fazer apenas com o HTML e CSS.
